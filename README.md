# Vengadesh — Personal Portfolio Website

A clean, multi-page personal portfolio website for a Power Platform Developer. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks required.

---

## Pages

| File | Page | Description |
|------|------|-------------|
| `index.html` | Home | Hero section with name, role, and stats |
| `about.html` | About | Bio, skills, experience, education tabs |
| `pro.html` | Projects | Project cards with category filter |
| `cer.html` | Certificates | Microsoft certification showcase |
| `con.html` | Contact | Contact form and details |
| `style.css` | Styles | All styles for every page |

---

## Features

- Responsive design (mobile, tablet, desktop)
- Fixed navbar with active page highlight
- Typing animation on hero name
- Project filter by category
- Skill progress bars
- Working contact form with success message
- Dark theme on Projects, Certificates, and Contact pages
- Light theme on About page

---

## How to Use

1. Download all files into one folder
2. Add your images inside an `images/` folder
3. Open `index.html` in any browser — no server needed
4. Update your name, bio, projects, and contact details directly in each HTML file

---

## Customization

- **Colors** — change `--pink` in `:root` inside `style.css`
- **Font** — replace the Google Fonts `Poppins` link in each HTML `<head>`
- **Content** — edit text directly in each `.html` file
- **Images** — replace `images/bcg.png` with your own photo
